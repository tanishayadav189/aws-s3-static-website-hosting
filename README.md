# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates how to host a static website using Amazon S3. 
A simple HTML website is deployed and made publicly accessible using S3 bucket policies and static website hosting.

## Features
- Static website hosting using S3
- Public access configuration
- Custom error page setup
- Simple and clean HTML structure

## Technologies Used
- Amazon Web Services (S3)
- HTML

##  Steps Performed
1. Created an S3 bucket
2. Disabled block public access
3. Uploaded index.html and error.html
4. Enabled static website hosting
5. Configured bucket policy for public access
